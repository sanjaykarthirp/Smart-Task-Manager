# Smart Task Manager

A full-stack MERN (MongoDB, Express, React, Node.js) application for managing tasks.

## Features
- User Authentication (Register/Login)
- Create, Read, Update, Delete (CRUD) tasks
- Responsive Design

## Tech Stack
- Frontend: React, Vite, Context API
- Backend: Node.js, Express, MongoDB
- Authentication: JWT

## Getting Started

1. Clone the repository
2. Run `npm install` in the root directory, `client`, and `server` directories.
3. Create a `.env` file in the `server` directory with your `MONGO_URI`, `JWT_SECRET`, and `PORT`.
4. Run `npm run dev` in the root directory to start both the frontend and backend.
